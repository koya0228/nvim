### Neovim ([Neovim:Home](https://neovim.io/))
<details>

```

```
</details>
<details>
<summary>macOS</summary>
```
$ brew install neovim
```
</details>
<details>
<summary>Ubuntu</summary>

```
sudo apt-get install neovim
```
</details>
