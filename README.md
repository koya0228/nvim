# NeoVim
## インストール
### Neovim ([Neovim:Home](https://neovim.io/))
<details>
<summary>WindowsOS</summary>
  
```
winget neovim.neovim
```
</details>
<details>
<summary>macOS</summary>
  
```
$ brew install neovim
```
</details>
<details>
<summary>Ubuntu</summary>

```
sudo apt-get install neovim
```
</details>
